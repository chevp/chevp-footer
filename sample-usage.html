ChevpFooter Web Component

A reusable footer component with customizable branding

Basic Usage

Include the script from CDN and use the component:

<!-- Include the web component -->
<script type="module" src="https://chevp.github.io/chevp-footer/chevp-footer.js"></script>

<!-- Use the component -->
<chevp-footer></chevp-footer>

Live Demo - Basic

Basic Usage

The footer will appear in the bottom-left corner with default settings.

Custom Avatar and Username

Customize the avatar and username:

<chevp-footer
    avatar="https://avatars.githubusercontent.com/u/78725490?s=48&v=4"
    username="YourGitHubUsername">
</chevp-footer>

Attributes Reference

Attribute Type Default Description
avatar String (URL) chevp's avatar URL to the avatar image
username String "chevp" GitHub username to display and link to

Features

  • Self-contained web component with Shadow DOM encapsulation
  • No external dependencies required
  • Responsive design with light/dark mode support
  • Customizable avatar and username
  • Links to GitHub profile
  • Fixed positioning in bottom-left corner
  • Smooth transitions and hover effects
